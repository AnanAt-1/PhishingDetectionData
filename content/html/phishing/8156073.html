Main Features

LiteSpeed Web Server is a drop-in replacement for Apache, and can load Apache configuration files directly. It can be used with any control panel that was designed for Apache, including cPanel, Plesk, and DirectAdmin. LiteSpeed is not a frontend proxy, but a complete replacement, featuring more efficient engines for popular Apache features, like Rewrite and ModSecurity. Replace Apache with LiteSpeed Web Server in less than 15 minutes with zero downtime and litte-to-no need for retraining!

Frequently Asked Questions

 What is LiteSpeed Web Server?

LiteSpeed Web Server is a drop-in Apache replacement and the leading high-performance, high-scalability server from LiteSpeed Technologies. You can replace your existing Apache server with LiteSpeed without changing your configuration or operating system details. As a drop-in replacement, LiteSpeed allows you to quickly eliminate Apache bottlenecks in 15 minutes with zero downtime. LiteSpeed's expansive range of features and easy-to-use web administration console are an asset to any effective web hosting infrastructure.

Three different types of licenses are available for LiteSpeed Web Server: Free Starter, Site Owner and Web Host. Please visit our LiteSpeed Web Server pricing page to choose the best license for your needs.

 How will my system benefit by using LiteSpeed Web Server?

Depending on specific server setup, each system may benefit differently. Overall, our customers tell us they enjoy significant decreases in server load and measurably faster site loading times. Other benefits include increased security, reduced hardware costs, ease of upgrade from Apache, and increased customer satisfaction. Plus, LiteSpeed’s caching plugins for the major CMS’s (WordPress, Drupal, Joomla, PrestaShop, Magento, OpenCart and more) make speeding up your sites a breeze.

 Is LiteSpeed Web Server Secure?

LiteSpeed takes security seriously. The Web Server is mod_security compatible, and comes equipped with cutting-edge anti-DDoS features to stop attackers before they cause serious problems.

 Is LiteSpeed Web Server Cost Effective?

When you switch from Apache to LiteSpeed Web Server, you do incur the cost of a license, but this is typically far below the expense of the hardware upgrade that would be required to get the same performance improvements under Apache.

 How often does LiteSpeed Web Server come out with new versions?

We don’t have a set time between each version, but we do like to keep on top of cutting edge technologies so we release new versions often. Check out our release log.

 Does LSWS work on VPS’s?

Yes, you can use LiteSpeed Web Server on a VPS.

 How does LiteSpeed Web Server compare to Apache and nginx?

LiteSpeed Web Server was built from the ground up to encorporate the best of both worlds: an event-driven architecture like nginx, and Apache .htaccess compatibility.

Being event-driven, LiteSpeed Web Server consumes fewer resources than Apache's process-based model, leading to better performance. LSWS has the ability to serve thousands of clients simultaneously with minimum memory and CPU usage. Sudden traffic spikes are no problem for LiteSpeed Web Server. And because LiteSpeed can read Apache configuration files directly (unlike nginx), LiteSpeed Web Server is thus compatible with control panels written for Apache. Switching from Apache can be done in 15 minutes.

For more info, please visit our Compare LiteSpeed Web Server vs. Apache vs. nginx page.

 Does LiteSpeed Enterprise work with CyberPanel?

Yes, LiteSpeed Enterprise does work with CyberPanel. Please see the CyberPanel Guide for more information, or order the LSWS CyberPanel bundle to get started.

 What control panels is LiteSpeed Web Server compatible with?

LiteSpeed Web Server can be used with any Apache-compatible control panel, such as cPanel, Plesk and Direct Admin.
