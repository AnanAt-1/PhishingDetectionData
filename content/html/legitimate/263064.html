Name: emacs
Group Type: Official GNU software

This software is part of the GNU Project.

Emacs is the extensible, customizable, self-documenting real-time display editor.

For information about Emacs, visit our webpage.  The development sources are hosted here on Savannah.

You can clone the main Git repository with:

git clone https://git.savannah.gnu.org/git/emacs.git


Detailed instructions, including how to clone GNU ELPA and NonGNU ELPA, are available on the Git repositories page.

For developer access, read these instructions.
