Lightwave Tools

Animation Loader
Loads Poser animation and bvh files and Mimic files into LW!
Conformer
Conform boned figures together in LW. Make your clothes behave!
CR2 Loader
CR2 Rigger
Directly import Poser figures from their CR2 file, includes all morphs and textures. Rig in Layout with complete parented skeleton and props. For LW use and creation of new Poser morphs. Now supports Poser 5 and its compressed files.
Load Obj as Morph
Load any exported obj file as a LW morph.
Export / Load Poses
Export poses to a file so they can be easily reloaded onto the same or different figures.
Write Morphs
Output a set of endomorphs as separate .obj files for creating Poser morph targets.
Write Morphs by Part
Create a single body level endomorph and have it output as separate .obj files for each body part for creating Poser morph targets.
Mirror Morphs
Create a morph on one right or left named part of a figure and have it automatically created, reversed, on the opposite part.
Export / Load Endomorphs
Export endomorphs to a file so they can be easily reloaded onto different figures.
Shade Weights
Auto shade Weight Maps
Write Weights
Load Weights
Saves the current weight map set to a text file and restores..
Map Game Export
Export 3dGS map and Wad files from LW
MDL7 Game Export
Export 3dGS mdl7 animated files from LW

Greenbriar Tools

ToolBox I, II, III,IV
3D Editors for creating Game and Poser content. Versions for game level building, animated model construction and export and game cut scene and stand alone animations.

GRRen - Greenbriar Poly Renderer
Film quality Renderer designed to work withToolBox. Includes special Toon, Auto Shadows and custom particle modes.

Acclaim Mocap Tools for Maya, C4D and ToolBox
Import, Edit and Export Acclaim ASF/AMC files. Convert BVH to Acclaim. Maintains compatibility with Massive animation files.

Cinema 4D Tools

CR2 Loader for C4D
Directly import Poser figures from their CR2 file, includes all morphs and textures. Rigs with complete parented skeleton and props.

Export Morph to .OBJ
Export Morph Tags and Polygon morphs to Poser compressed morph format. Splits body morphs into Poser body part morph targets.

Morph System for C4D
Morph Controller to go with CR2 Loader that gives full morph functionality with C4D Basic!

Conformer for C4D
