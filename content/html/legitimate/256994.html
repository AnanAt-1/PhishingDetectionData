Google Groups Simplifier

bleb.org > Tools > Google Groups URL simplifier

Google has made available archives of Usenet dating back to 1995. However, it is still lacking a "view for bookmarking" feature that DejaNews used to provide.

Alternatively, you can use my Google random jump for an exciting journey into the unknown ;-)

Simply paste a Google Groups URL below and it will be simplified.



To add a button to your browser which will take the current page and automatically submit it here, the JavaScript is:

javascript:location.href='http://bleb.org/google?url='+escape(window.location);

Alternatively, select the following link and add it to your bookmarks: Simplify
