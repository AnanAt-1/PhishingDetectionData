Mar 30, 2015  UltraMon 3.3.0 released
UltraMon 3.3.0 is now available for download. New in this release: new taskbar extensions feature for Windows 8, and new window button compatibility options which improve support for applications with custom window title bars. Several bugs have been fixed as well, for details see the release notes. This is a free upgrade if you have a version 3 license.

Dec 25, 2012  UltraMon 3.2.2 released
UltraMon 3.2.2 is now available for download. New in this release: the window buttons now match the look of the native buttons on Windows 8, and several bugs have been fixed. See the release notes for more information.

This is a free upgrade if you have a version 3 license.

UltraMon 3.2.0 is now available for download. This release has support for Windows 8, as well as several bug fixes, see the release notes for more information.

Dec 20, 2010  UltraMon 3.1.0 released
UltraMon 3.1.0 is now available for download. This release has new display configuration features such as screen orientation change, output switching and support for legacy drivers on Windows 7, new scripting features (collection of all open windows, Windows 7 display configuration features) and several bug fixes, see the release notes for more information.

Important notice for system administrators: if you have UltraMon installed on a terminal server, make sure to take a look at the release notes, there was a licensing change which may affect you.
