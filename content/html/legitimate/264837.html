This is the home page for the Texttools. The Texttools packages are a GPL, ncurses-based library for the Linux console. Texttools contain more than 600 procedures and functions to create windows, draw scroll bars, handle the mouse and keyboard events, play sounds, and much more. The Texttools package also provides a thick binding to Linux kernel calls. You can create a wide variety of application programs using Texttools alone.

More than 600 subroutines and a variety of window objects including scroll bars, selection lists and text edit boxes.

Texttools has event queues and supports mice just like X Windows but is much easier to program.
