Lecture slides and activities on XMLRPC

In 2003 I gave a lecture on Distributed Computing at the University of Luton. Approximately two weeks have been devoted to XMLRPC and for that I produced a number of slides and activities.

Just in case that somebody else finds this material helpful here it is.

  • Introduction into XMLRPC: intro.ppt (PowerPoint) intro.pdf (pdf), html.. (and here a slightly extended version translated into Vietnamese).
  • The XMLRPC specification: spec.ppt (PowerPoint) spec.pdf (pdf), html.
  • Activity 1: Hello World.
  • Activity 2: A simple chat.
  • The assignment (on CORBA and XMLRPC): word, pdf, html; with marking scheme: word, pdf, html.
  • Some miscellaneous Resources on XMLRPC.

The Java package cis69mc.jar mentioned in the activities is identical with the package provided at http://xml.apache.org/xmlrpc with the single addition that the class uk.co.wilson.xml.MinML has four additional lines. Here is the modified version of MinML.java and the rest of the source code is here: xmlrpc-1.2-b1-src.tar.gz (for the latest version please see directly the apache repository.)
