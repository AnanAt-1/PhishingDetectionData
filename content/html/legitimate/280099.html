A Great Grabber

A Great Grabber is a software to scan, search and grab the Internet. All text based webpages, like html, htm etc. are scanned for needed files. You input a start webpage, the extensions of the files where you are interested in and let the software searching and downloading these files. You could additional retrieve all found email addresses from the webpages. While running, all downloaded images are immediately shown in the image viewing window, all vidoes in the video window and all audio files are played. So you could lay back, let the program scan the Internet and view all found images or videos one after another. You could download all filetypes you like! By example: .JPG, .GIF, .MP3, .AVI, .PNG, .PDF, .WAV.

Startwebpage - Input one or several webpage addresses from which the software starts to scan.
Depth - Select the depth of links, from which the webpages should be scanned.
File Extensions - You can search for all files you want. Input the extensions which should be included into the search.
Email Addresses - The software searches for included email addresses and saves them into a file if selected.

 

Images

All downloaded images are shown in the image viewing window after download. So you can lay back and watch the downloaded images while scanning the web. The images are shown in a previous selected interval. All incoming images are inserted into a queue, from which the images are shown.

 

Videos

All downloaded videos are shown in the video viewing window after download. So you can lay back and watch the downloaded videos while scanning the web. The videos are shown one after another, everytime a new video has downloaded.

 

Audio Files

All downloaded audio files are played after download. So you can lay back and listen to what you have found in the web. All incoming audio files are inserted into a queue, from which they are played.
