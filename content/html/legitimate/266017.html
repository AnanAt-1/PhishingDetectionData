  DAP Pro 2.4


Look into a new window to quality!


 DAP Pro is a unique tool for run-time debugging, testing and investigation of your applications. It is based on Delphi and C++ Builder RTTI controlling and can be very helpful for programmers and Quality Assurance specialists. Other debugging tools use code hooking to allow you investigation of applications and you should know approximately, where the problem is. DAP enables you to run the application and get all component and class information at any moment you want. You know that the most part of modern applications on Delphi and C++ Builder are created to use all power of VCL component technology. This means that first of all debugging and testing tools should support components. DAP Pro is designed to realize this general idea.
You can look at DAP Pro screenshots to get more details how to use this powerful tool in your day-to-day activities.

  DAP Lite 1.3


Freeware tool for application investigation

DAP Lite is a feature limited version of DAP Pro. It is absolutely freeware and can be used for getting general information on components in your applications: a class name, a parent, etc. You can distribute DAP Lite with your applications for your beta-testers. Using DAP Lite they will be able to send you more accurate and informative reports!

You can look at DAP Pro and DAP Lite screenshots to get more details on how to use these tools.
