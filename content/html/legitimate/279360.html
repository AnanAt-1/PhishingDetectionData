SLS oss657a: Minimum PIT Count Supplement

Date posted: 23-APR-03

For: OpenServer Release 5.0.6

Description: This patch addresses the following:

Some systems running SCO OpenServer 5.0.6 on CPUs over 1 gigahertz may experience periodic system pauses that can be resumed by console keyboard activity. The problem is caused by the Programmable Interrupt Timer (PIT) being programmed to assert and deassert itself so fast that the CPU's attempt to service the interrupt is unsuccessful. This leads to an endless loop of spurious interrupts which can only be broken by a legitimate interrupt (such as a keystroke on the console keyboard).

SLS oss653a: Initial VGA Recognition Supplement

Date posted: 26-MAR-03

For: OpenServer Release 5.0.2 - 5.0.7

Description: This supplement addresses the following problem:

Some systems incorrectly initialize the Primary Display Type bits in the CMOS Equipment Byte (0x14). This can cause OpenServer to incorrectly identify the video as CGA instead of VGA. OpenServer relies on reading the Primary Display Type bits to determine which video driver to use.

This can cause a variety of symptoms including:

  • Black or Blank screen
  • Invisible or unreadable fonts
  • Half letter fonts
  • Squished or Scrunched screen

SLS oss605a: NetBIOS Drivers Supplement

Date posted: 04-OCT-99 For: SCO OpenServer Release 5.0.4, 5.0.5
