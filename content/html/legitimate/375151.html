FuSa GamePad Version 0.3
6:09 PM

Perhaps a simpliest way to turn your psp into a powerful PC gamepad (and yes PS3 too).
  • Just launch this tiny homebrew
  • Connect your psp via USB cable to PC (PS3)
  • And you are ready to go :)

NEW FEATURES:

  • Updated driver core
  • Adjustable configuration (via config.ini)
  • Led indicating
  • Digital/Analog modes
  • Deadzone support
  • No conflicts with drivers (No more "PSP type D" - now it's "FuSa GamePad")

If you have any problems with this gamepad, make sure...

1.) You disable the remotejoy plugin (look in your PSP's plugins directory and delete remotejoy & its corresponding line in the config file).
Because if you don't disable remotejoy, fusa will stay stuck at 'connecting' when you launch it

2.) Your PSP is on CFW 3.80 or later
Because I tried this on 3.71m33-2 and while windows recognized the fusa gamepad, it wouldn't register any button presses.
I updated to 3.80 m33-5 and now fusa works perfectly with my PC. (Newer cfws work too, I chose to stay on 3.x)

Note the following cfws which are confirmed to work:
3.80 m33-5
6.39 PRO-B10
6.20 TN-D w/ TNHE
5.50 Gen D3 Final
5.00 m33-6

Users have noted that FuSa does not work perfectly with the PS3. There are some workarounds.
See this thread: http://foosa.do.am/forum/6-180-2

Note that you can rebind the psp buttons (search joytokey if you're on windows)

Note that v0.1 of fusa gamepad is reported to be working with android while .2 and .3 apparently don't.
