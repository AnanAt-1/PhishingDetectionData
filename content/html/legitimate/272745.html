Features

Active Directory Single Sign-On

Allows users to access VisualSVN Server using their current Active Directory domain credentials. Secure Kerberos V5 or NTLM authentication protocols are used. Support for two-factor authentication and smart cards is available.

Learn more

Multisite Repository Replication

Provides high-performance replication between geographically distributed sites using VisualSVN Distributed File System (VDFS) technology. Distributed repositories are writable and functionally equivalent to regular Subversion repositories.

Learn more

Full-Text Search

Search through the contents and history of your repositories — in any folder, at any revision. The search engine offers high performance, continuous indexing of new revisions and has virtually no limits on the repository sizes.
