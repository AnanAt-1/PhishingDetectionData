FTP Navigator

FTP Navigator is an easy-to-use FTP client for Windows that simplifies transferring files over the internet. Its user-friendly interface allows you to easily navigate remote file directories and perform basic file operations like upload, download, delete, and rename for files and folders. The application includes a transfer progress window and can automatically resume incomplete transfers or maintain a stable connection with the FTP server.

For advanced users, FTP Navigator offers directory syncing, extensive command-line controls, and the ability to manually enter FTP commands. The software also automatically sets file transfer parameters and logs all actions for future reference. A list of public FTP sites is provided for quick access. FTP Navigator auto-corrects user errors and supports multiple input and setting formats, making it a versatile solution for internet-based file transfers.


Supported formats:

- ftp.hostname.com

- ftp://ftp.hostname.com

- [email protected]/folder

- ftp://[email protected]:port/folder

- ftp://[email protected]:port/folder

- ftp://[email protected]

- ftp://[email protected]:port

- http://ftp.softwarea.com


The software's real-time correction feature removes extra spaces or invalid characters during the connection process.

FTP Navigator is compatible with various server types and allows customization of server response formats. It lets you define rules for interpreting server-hosted files and folders, save commonly-used connections for one-click access, and remembers default directories for local and remote systems.

Note that file names with different cases are treated as distinct files on the server. To avoid issues, the software includes a feature that lets you modify the case type on your local machine and remote servers.

Customization options are available in the "Program Settings" menu, where you can tailor screen attributes and menu choices. The program menu is context-sensitive and displays options applicable to the current operating mode. For special commands not available in the program, you can access "Custom Commands" and save them for future use.
