JustInTime 0.3.2 Information Page
Introduction
JustInTime is a helper application allowing to set the Palm time from a host running UNIX (or any other platform capable to run pilot-xfer).
Whats new?
A patch against pilot-link 0.9.5 and a standalone version called pilot-justintime (for 0.11.x) is included in the source distribution. There is a JPilot plugin available at:
http://www.klawitter.de/palm/JustInTime-JPP-03.tgz
General Usage

Once the program is installed, you do not need to worry about it any more.

Whenever you start the changed version of pilot-xfer with the option -t, pilot-xfer will compute the time difference between your Palm and the host and store this information in a small database on the Palm.

If you are using JPilot and have the JustInTime JPilot Plugin installed, you don't need the pilot-xfer stuff.

Installing this database will notify JustInTime, which itself again starts an (invisible) alarm a few seconds later. During this alarm the computed difference will be added to the current time and the now obsolete database will be deleted. That's all, folks.

Known Bugs

Sometimes the HotSync Manager crashes after the reception of a new time. Still have to figure that out...

When the HotSync Button is pressed after starting pilot-xfer, pilot-xfer sometimes aborts before connecting properly. (Workaround: press HotSync before starting pilot-xfer as with POSE :-)
