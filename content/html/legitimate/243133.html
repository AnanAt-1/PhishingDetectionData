INTRODUCTION

The information contained in this guide is basic to most conventional remodeling and addition projects. The reader is also referred to the current editions of the following:

  • 1994 ICBO Uniform Building Code (UBC)
  • 1994 ICBO Uniform Plumbing Code (UPC)
  • 1994 ICBO Uniform Mechanical Code (UMC)
  • 1995 National Electrical Code (NEC)
  • California Administrative Code (Title 24)
  • Local Zoning and other ordinances.

When conflicts occur between the different codes regarding a certain item, the most restrictive code section applies.

Good luck, and happy hammering!
