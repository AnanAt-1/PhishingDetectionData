  • Select Images, Illustration, Footage or All in pull down next to the search bar.
  • A search for 'Animals Running' will match both terms. There is no need to add "+" (Animals + Running).
  • The pipe character '|' will change this search ("Animals | Running") to "Animals" OR "Running".
  • The minus '-' chacter will remove a keyword from the search, 'Animals Running -horse' will remove horse from the search.
  • If you're having trouble locating an image or clip, click on "Request a Search" at bottom of page, and we'll be happy to assist.
Results Options:
  • Select the number of images returned in each page (20, 50, 100, 200).
  • Select "Newest" or "Relevancy" as sort order. "Relevancy" uses multiple metrics to determine order by frequency of metadata.
  • Select "Classic View" for more information surfaced for each image. "Gallery View" presents just images in a mosaic format.
  • If you have registered and are logged in, you may save your search result options in the pull down under your email address (User Profile).
Search Lock:
  • By selecting the lock icon to the left of the collection indicator, you can restrict all searches to one of three options.
  • My Assets - Your searches will only surface images you can download within your active Pack, Subscription, or items you have already licensed.
  • Premium - These are the core assets of the site available as RF or RM licenses.
  • Subscription - This is media available as a lower-cost option at a single price point for multiple downloads.
  • If you deactivate search lock, all media will be searched by default.
Using Filters:
  • The filters button at left of page provides additional options to add more specificity to your search.
  • As you add filters, the filter ID will appear above search results. To clear a filter, click the "X".
  • The contributor filter will list all artists represented in your search. You can select the specific artist's work you'd like to see.
Page Navigation:
  • The arrow keys on your keyboard may be used to scroll pages and move between pages.
  • Page numbers may be entered in navigation box in the footer.
