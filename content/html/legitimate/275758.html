Introduction

Published: Sep 28, 2006. Updated: Apr 7, 2010.

The best Firefox bookmarks sorter!

Alphabetically sort your Firefox bookmarks.html file with this easy to use Java program.

Features

  • Free!
  • Fast sorting
  • Automatically backs up your bookmarks.html file before sorting
  • Remembers where your bookmarks.html file is located
  • Works on all major operating systems
  • Also sorts Mozilla and Netscape bookmarks

Testimonial

"Great, simple utility that fixes a big omission in Firefox. It's brainlessly simple - when you start it and click File>Open, it guides you through the relevant files. It's easy to find Mozilla, Firefox and then the Bookmarks file. Then select the particular profile (usually the Default one) and PRESTO! That annoying hodgepodge of bookmarks and folders in the dropdown menu is Alphabetical!"  --Pat R. via email
