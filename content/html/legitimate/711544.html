Local Area Network (LAN) Settings

The Local Area Network (LAN) Settings dialog box is displayed when you click the LAN Settings button on the Select SDLX Server dialog box.

 

The Local Area Network (LAN) Settings dialog box is described below.

 

Field

Description

Use the system default settings for proxy server

This is the default setting.

Override the system default settings with these settings

Select this to specify non-default proxy server settings.

Use a proxy server for your LAN

Select this to activate the Address and Port boxes.

Address

Enter the proxy server address.

Port

Enter the number of the port to be used.
