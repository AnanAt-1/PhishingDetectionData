Otter: Tool for Topology Display

Developed in 1998, Otter is a historical CAIDA tool used for visualizing arbitrary network data that can be expressed as a set of nodes, links or paths. Otter was developed to handle visualization tasks for a wide variety of Internet data, including data sets on topology, workload, performance, and routing. Otter is in maintained and continues to be supported by, but is no longer actively developed by CAIDA.
