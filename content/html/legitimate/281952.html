RISC OS Software

Learn about RISC OS Computers with RISC OS operating system and the ARM processor.
  • Nested Window Manager: An improved version of the official document (with better description on the contents of R4 on entry for Wimp_OpenWindow). Based on the original.
  • BasText: A tool to convert tokenised BASIC to Text and vice versa
  • OLE Specifications: An attempt at clear documentation for the OLE protocol.
  • !ShowRes: Show the contents of a Resource file as text. Can also produce a file in the style of the Message file as produced by ResEdit, with lots of comments that describe the entries.
  • Calendar: A module to convert dates from one calendar to another, and to make calculations with dates in any of the calendars.
  • !TempTEd: A Template Text Editor. Export texts from template files, modify them with your favorite editor, and merge them back into the template file.
  • MalCheck: A tool to check for problems with dynamically allocated memory blocks in C.
  • !DiffTB: A diff tool with throwback. Based on a hack by Druck.
  • Revision control (RCS v5.7.1.2): indispensable tool for software developers.
  • GNU diff and diff3 for RISC OS, with built-in file name conversion.
  • !Spin: Harddisk spin up/down control. Supports multiple disks.

Note

For 32-bit () ready software, you may need an update of some of the modules in your !System directory. The best place to get them is from the Iyonix site.
