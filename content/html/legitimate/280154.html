Web Publisher
Web Publisher is an automated FTP client that allows you to upload and then update your web site easily. Web Publisher can automatically find and upload new and modified files with one mouse click. Old folders and files may be removed automatically with confirmations.

Key features

  • upload your local web site copy to the FTP server
  • automated updating of new and modified files
  • removal of old folders and files
  • smart FTP compare
  • safe uploading (allows users to access web site while uploading)
  • filtering of files by directory, name or mask
  • automated setting of FTP permissions
  • support of ftp proxy and socks server
