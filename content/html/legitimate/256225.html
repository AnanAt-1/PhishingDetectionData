Using the table below you can break down the various metrics of domains we have on offer. Metrics are coming from Moz (PA/DA), MajesticSEO (CF/TF) and Ahrefs( BL/DL). PA should ideally to aim for 30+, though PA 29 is not terrible, PA 28 is good, and you’ll want to go for PA 25+ on that (We only sell lower if there is a decent sum of live links indexed in ahrefs that are not shown in Moz).

In the column “BL+” we list “BL from” E = .edu or .gov; Y = Yahoo Directory; D = DMOZ; W = Wikipedia; A = answers.com; B = BBC; C = CNN; N = Newspaper

The metrics include the CF / TF ratio from MajesticSEO, which is the only metric that can’t be manipulated and thus allows to quickly identify SPAMMY domains (read more). The closer this value is to zero, the better the URL is. We only include domains to our lists with no SPAM and a CF/TF ratio lower than 1.8.  If it’s anywhere above that, we don’t list the domain.
