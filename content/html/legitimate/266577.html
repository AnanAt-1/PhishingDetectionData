The MPFUN multiprecision software files have moved and are currently maintained here:

http://crd.lbl.gov/~dhbailey/mpdist/ 

1. mpfun90 package:

    For:  multiple precision arithmetic, Fortran-90:

    By:  David H Bailey 

    Zip file:   http://crd.lbl.gov/~dhbailey/mpdist/mpfun90.tar.gz

 

2.  mpfun77 package:

     For:  multiple precision arithmetic (use mpfun90 if possible)

     By:   David H Bailey 

     Zip file:  http://crd.lbl.gov/~dhbailey/mpdist/mpfun77.tar.gz

 

3.  arprec package:

     For:  multiple precision arithmetic, C++ and/or Fortran-90

     By:  David H Bailey, Xioaye Li, Brandon Thompson

     Zip file:  http://crd.lbl.gov/~dhbailey/mpdist/arprec.tar.gz

 

4.  qd package:

     For:  double-double (32 digits) or quad-double (64-digits) arithmetic, C++ or Fortran-90

     By:  David H Bailey, Xiaoye Li, Yozo Hida

     Zip file:  http://crd.lbl.gov/~dhbailey/mpdist/qd.tar.gz
