HOW TO SET FAVOURITES

With this option you can compile a list of your favourite channels. These channels are displayed at the very top of the standard list of channels on your UPC TV Box or your Horizon HD Recorder.

UPC TV BOX

If you want to create your own favourites / channel list, you have to set up a profile.

To create a profile, press the Profile button on your remote control and follow the instructions on the screen. With the same button you can switch between your profiles at any time.

In your profile, you can create your own channel list so you can save favourites and sort the channels freely.

Heres how:
 
  1. "Menu" > "Settings" > "Profiles" > "Manage channels".
  2. Here you can use Add channel to mark the channels as favourites and save them in your own channel list. 

HORIZON HD RECORDER

You can create a list with your favourite channels under: " Menu" > "Toolbox" > "Preferences" > "Favourite channels".

 

Set Favorites:

 

  1. Access the "Menu" with the Horizon remote control.
  2. Select "Preferences" under "Toolbox". 
  3. Then select "Favourite channels" and then "Set up favourites".
  4. While scrolling through the available channels mark your favourites with the "OK" button.
  5. Once a channel is selected, there is a symbol next to the channel name.   

Important: Confirm your favourites list with Enable favourites mode to save your favourite channels.


Reorder favourites:

 

  1. Access the "Menu" with the Horizon remote control.
  2. Select "Preferences" under "Toolbox".
  3. Then select "Favourite channels" and then "Set up favourites".
  4. In the favourites list select the channel you want to move with the "OK" button.
  5. Then use the "Up" or "Down" keys to move the channel to the desired position.
  6. Confirm the new position with the "OK" button.

Important: Confirm your reordered favourites list to save it.

 

MEDIABOX

Setting up favourites:
 
  1. To set up your favourites, first open the Electronic TV Guide by pressing the GUIDE button.
  2. Press the green button to display the settings screen and select "Set up favourites", press "OK" to confirm. 
  3. Add favourites to your personal list by selecting the channels using the up/down arrows on the remote control.
  4. Press the blue button to add the selected channel to your favourites (providing this channel has not already been set as a favourite). A "smiley" indicates which channels have been set as a favourite.
