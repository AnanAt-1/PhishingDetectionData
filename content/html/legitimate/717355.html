Dear mops,

As most protected brands and trademarks the Compact Disc logo (which
is, by the way, owned by Philips) uses a non-standard font that has
been designed especially for the logo. However, a True Type font has
been created based on the logo's well-known charcaters. Would you be
interested in that?
Also, you can get the logo itself in PostScript and True Type format.
Would that help you?

Regards,
