Overview of XML, Web Services and Enterprise Messaging
Duration:  1 Day
Audience Prerequisites

This course is designed for professionals who are interested in learning about XML-based messaging and the related products, frameworks, architectures and best practices.

  • Managers and Executives
  • Consultants
  • Sales engineers
  • Messaging application developers
  • Software architects

None

    Description
    "Overview of XML, Web Services and Enterprise Messaging" is a one-day, instructor-led course which provides a high-level overview of open standards and technologies in the areas of enterprise messaging services, middleware architectures, enterprise integration, security and best practices. Upon completion of this training course, attendees will become familiar with the latest XML standards, Web Services, Java 2 Enterprise Edition (J2EE), Service Oriented Architecture (SOA), Java Messaging Service (JMS) and some popular middleware systems.
    This lecture course focuses on the latest technologies used in the enterprise communication spectrum.
    This is a lecture only course. The instructor will demonstrate some JMS-based applications on the instructor's computer.
    Objectives

    The main objective of this course is to familiarize the attendees with XML messaging and all the middleware technologies that are required to understand Cisco System's Application-Oriented Networking (AON) product. However, this course does not cover any topics of the AON product. This course teaches developers a variety technologies that are useful in developing XML messaging based applications in a SOA framework. On completion of this course, attendees will be able to understand:

    • XML technologies like Schemas, Style Sheets, and XPath and XML Parsers
    • The concepts of Web Services technologies: SOAP, WSDL, UDDI
    • Service Oriented Architecture (SOA)
    • The latest XML security standards
    • The latest Web Services security standards
    • The J2EE framework
    • JMS-based messaging using WebSphere MQ and Tibco as messaging providers
    Course Outline

    Module 1

    Module 2

    Module 9

    Module 10

    • Overview of Enterprise Messaging with JMS: Using WebSphere MQ and Tibco as providers
