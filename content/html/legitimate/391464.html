Glossary
The 1984 Topps baseball card set consisted of 792 baseball cards that measure 2-1/2" by 3-1/2"  

Key Cards:

  • 8 - Don Mattingly RC
  • 17 - Nolan Ryan 
  • 182 - Darryl Strawberry RC
  • 490 - Cal Ripken Jr

Glossary:

  • CL - Checklist
  • CO - Coach
  • COR - Corrected Card
  • DP - Double Print  
  • ERR - Error card
  • FTC - First Topps Card
  • LL  - League Leaders 
  • MG - Manager
  • RB - Record Breaker
  • RC -  Rookie Card
  • SP - Single or Short Print
  • TL - Team Leaders
  • UER - Uncorrected Error
