Get the latest hardware development kits and tools

  • Windows Driver Kit

    The Windows Driver Kit (WDK) is used to develop, test, and deploy Windows drivers. Download the latest public version here or join the Insider Program to get access to insider builds.

  • Windows Hardware Lab Kit

    The Windows Hardware Lab Kit (HLK) is used to validate hardware and drivers for Windows compatibility. Download the latest public version here or join the Insider Program to get access to insider builds.

  • Windows Assessment and Deployment Kit

    The Windows Assessment and Deployment Kit (ADK) is used to customize, deploy and benchmark Windows 10 images. Download the latest public version here or join the Insider Program to get access to insider builds.

  • Windows Debugging Tools

    The Windows Debugger (WinDbg) can be used to debug kernel and user mode code, analyze crash dumps and to examine the CPU registers as code executes. Download the latest public version here or join the Insider Program to get access to insider builds.
